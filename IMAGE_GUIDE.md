# Image Upload Guide for Nepali Headbandz Game

Upload your images to the `images/` folder with these exact filenames.

**Image Requirements:**
- Format: JPG, PNG, or GIF
- Recommended size: 400x400 to 800x800 pixels
- The game will automatically fall back to emoji if an image is missing

## How to Add Images

1. Find or download images for each item
2. Rename them to match the **Filename** column exactly
3. Save them in the `images/` folder
4. Refresh the game - images will appear automatically!

**Note:** If an image file is missing, the game will show an emoji instead. This means you can add images gradually!
